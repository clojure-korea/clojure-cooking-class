# Leiningen 템플릿으로 프로젝트 만들기

by Eunmin Kim

## 이럴 때 쓰세요

다른 사람이 만든 Leiningen 프로젝트 템플릿을 사용하고 싶습니다.

## 어떻게 하나요?

### 1. 프로젝트 생성하기

`lein new 템플릿 프로젝트명`으로 프로젝트를 생성하면 됩니다.

```bash
$ lein new compojure hello-world
$ cd hello-world
$ lein ring server-headless
```
